 Bookstore Application
A modern, dynamic e-commerce interface for browsing and managing books. This project was developed to demonstrate proficiency in building single-page applications (SPAs) using React, focusing on state management, dynamic routing, and responsive design.

🚀 Live Demo
Link: https://anilnxtwaveproject1.vercel.app/

✨ Key Features
Dynamic Book Catalog: Browse a wide collection of books with real-time updates.

Category Filtering: Easily filter books based on genres or categories.

Detailed View: Specific routes for individual book details to provide a deeper look at titles.

Responsive UI: Fully optimized for mobile, tablet, and desktop using modern CSS techniques.

Interactive Search: Quick search functionality to find specific books instantly.

🛠️ Tech Stack
Frontend: React.js

Styling: CSS

Routing: React Router DOM

Deployment: Vercel

📖 What I Accomplished
Implemented React Router for seamless navigation between the home page and product details without page reloads.

Managed application state effectively using React Hooks (useState, useEffect) to handle data fetching and UI changes.

Created reusable Functional Components to maintain a clean and scalable codebase.

Integrated an interactive UI that responds to user inputs for filtering and searching.

🏁 How to Run Locally
Clone the repository:

Bash

git clone https://github.com/aniljiA1/anilnxtwaveproject1.git
Navigate to the project directory:

Bash

cd anilnxtwaveproject1
Install dependencies:

Bash

npm install
Start the development server:

Bash

npm run dev
The app will open at http://localhost:5173.
